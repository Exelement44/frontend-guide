Что такое CSS?

CSS (Cascading Style Sheets) — это язык стилей, который определяет, как HTML-элементы должны отображаться на экране. Если HTML — это скелет страницы, то CSS — это её одежда, макияж и прическа.

Простая аналогия: Строительство дома.

HTML — это фундамент, стены, окна, двери (структура)

CSS — это цвет стен, обои, стиль окон, форма крыши (внешний вид)

JavaScript — это электричество, водопровод, лифт (интерактивность)

Без CSS все сайты выглядели бы как чистый текст с синими ссылками — как в 1990-х годах.

Базовый пример подключения CSS

Вот минимальный пример HTML-страницы со стилями:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя первая стилизованная страница</title>
    <!-- ПОДКЛЮЧЕНИЕ CSS -->
    <style>
        /* Здесь пишутся стили */
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
            margin: 0;
            padding: 20px;
        }
        
        h1 {
            color: #2563eb;
            text-align: center;
        }
        
        .card {
            background: white;
            padding: 20px;
            border-radius: 10px;
            box-shadow: 0 4px 6px rgba(0,0,0,0.1);
        }
    </style>
</head>
<body>
    <h1>Добро пожаловать в мир CSS!</h1>
    
    <div class="card">
        <h2>Это карточка со стилями</h2>
        <p>Раньше этот текст был обычным, а теперь он красивый!</p>
        <button>Кнопка с будущими стилями</button>
    </div>
</body>
</html>

Давайте разберем, как это работает.

3 способа подключения CSS

1. Встроенные стили (inline)

Стили прямо в теге через атрибут style:

<h1 style="color: blue; font-size: 32px;">Заголовок</h1>
<p style="background: yellow; padding: 10px;">Текст</p>

Плюсы: Быстро, для разовых стилей

Минусы: Нельзя переиспользовать, смешивает структуру и оформление

2. Внутренние стили (internal)

Стили внутри <style> в <head>:

<head>
    <style>
        h1 { color: blue; }
        p { background: yellow; }
    </style>
</head>

Плюсы: Все стили в одном месте, только для этой страницы

Минусы: Нельзя использовать на других страницах

3. Внешние стили (external)

Отдельный файл .css, подключаемый через <link>:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

Файл styles.css:

h1 { color: blue; }
p { background: yellow; }

Плюсы: Переиспользование на всех страницах, кэширование браузером

Минусы: Дополнительный файл

Рекомендация: Всегда используйте внешние стили для реальных проектов!

Селекторы: Как выбрать элементы

Селекторы — это способ сказать CSS: "Эти стили применяй к ЭТИМ элементам".

Базовые селекторы:

Селектор Пример Что выбирает
По тегу p Все параграфы <p>
По классу .card Элементы с class="card"
По id #header Элемент с id="header"
Универсальный * Все элементы

Примеры в коде:

/* Все заголовки h2 */
h2 {
    color: darkblue;
}

/* Все элементы с классом "important" */
.important {
    background: yellow;
    font-weight: bold;
}

/* Элемент с id="main-title" */
#main-title {
    font-size: 48px;
    text-align: center;
}

/* Все элементы на странице */
* {
    box-sizing: border-box;
}

Комбинированные селекторы:

/* Все параграфы внутри элементов с классом .article */
.article p {
    line-height: 1.6;
}

/* Все ссылки при наведении */
a:hover {
    color: red;
}

/* Первый элемент списка */
ul li:first-child {
    font-weight: bold;
}

Свойства и значения: Что можно менять

Текстовые свойства:

p {
    /* Шрифт */
    font-family: "Arial", sans-serif;
    font-size: 16px;           /* Размер: px, em, rem */
    font-weight: 400;          /* Толщина: 100-900 или normal, bold */
    font-style: italic;        /* Стиль: normal, italic */
    
    /* Цвет и выравнивание */
    color: #333333;            /* Цвет текста: #hex, rgb(), названия */
    text-align: center;        /* Выравнивание: left, center, right, justify */
    line-height: 1.5;          /* Межстрочный интервал */
    text-decoration: underline;/* Подчеркивание: none, underline, line-through */
}

Цвета в CSS:

/* Разные способы задать цвет */
.color-examples {
    color: red;                /* По имени */
    color: #ff0000;            /* HEX (шестнадцатеричный) */
    color: rgb(255, 0, 0);     /* RGB (0-255) */
    color: rgba(255, 0, 0, 0.5);/* RGB с прозрачностью (0-1) */
    color: hsl(0, 100%, 50%);  /* HSL (тон, насыщенность, яркость) */
}

Пример текстовых стилей:

<!DOCTYPE html>
<html>
<head>
    <style>
        .fancy-text {
            font-family: "Georgia", serif;
            font-size: 20px;
            font-weight: bold;
            color: #2d3748;
            text-align: justify;
            line-height: 1.8;
            letter-spacing: 1px;
        }
        
        .special {
            color: #e53e3e;
            text-transform: uppercase;
            text-shadow: 2px 2px 4px rgba(0,0,0,0.2);
        }
    </style>
</head>
<body>
    <p class="fancy-text">
        Это текст с примененными стилями CSS. 
        <span class="special">Обратите внимание</span> 
        на разницу в оформлении разных частей текста.
    </p>
</body>
</html>

Блочная модель (Box Model)

Каждый HTML-элемент — это прямоугольный блок. Блочная модель описывает, как устроен этот блок:

┌─────────────────────────────────────────┐
│              margin (внешний отступ)    │
│  ┌───────────────────────────────────┐  │
│  │         border (граница)          │  │
│  │  ┌─────────────────────────────┐  │  │
│  │  │      padding (внутренний    │  │  │
│  │  │          отступ)            │  │  │
│  │  │  ┌───────────────────────┐  │  │  │
│  │  │  │     content (контент) │  │  │  │
│  │  │  └───────────────────────┘  │  │  │
│  │  └─────────────────────────────┘  │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘

В качестве примера даже можно взять ту же кнопку "Следующий урок" где надпись - контент, внутренний отступ - padding, граница кнопки(в том числе сгругленные углы) - border, и наконец расстояние от кнопки "Предыдущий урок" - margin

Свойства блочной модели:

.box {
    /* Размер контента */
    width: 300px;
    height: 200px;
    
    /* Внутренние отступы (padding) */
    padding: 20px;                    /* Все стороны: 20px */
    padding: 10px 20px;               /* Вертикально | Горизонтально */
    padding: 10px 20px 30px 40px;     /* Верх | Право | Низ | Лево */
    
    /* Границы (border) */
    border: 2px solid #2563eb;        /* Толщина | Стиль | Цвет */
    border-radius: 10px;              /* Скругление углов */
    
    /* Внешние отступы (margin) */
    margin: 20px auto;                /* auto центрирует по горизонтали */
    
    /* Фон */
    background-color: #f8fafc;
    background-image: linear-gradient(45deg, #f3f4f6, #ffffff);
}

Важное свойство box-sizing:

/* Без box-sizing: border-box */
.old-box {
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    /* Фактическая ширина: 300 + 20*2 + 5*2 = 350px! */
}

/* С box-sizing: border-box (рекомендуется всегда!) */
.modern-box {
    box-sizing: border-box;
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    /* Ширина остается 300px! padding и border внутрь */
}

/* Лучше задать для всех элементов */
* {
    box-sizing: border-box;
}

Практическое задание: Создаем стилизованную карточку

Задача: Превратить обычный HTML в красивую карточку с помощью CSS.

Исходный HTML:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Практика CSS</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="product-card">
        <img src="https://via.placeholder.com/300x200" alt="Товар">
        <h2 class="product-title">Смартфон Flagship Pro</h2>
        <p class="product-description">Мощный смартфон с лучшей камерой на рынке</p>
        <div class="price">49 990 ₽</div>
        <button class="buy-button">Добавить в корзину</button>
    </div>
</body>
</html>

Что нужно сделать в файле style.css:

  1. Задать стили для всего тела страницы (body)
  2. Стилизовать карточку (.product-card)
  3. Оформить заголовок (.product-title)
  4. Стилизовать описание (.product-description)
  5. Сделать красивую цену (.price)
  6. Оформить кнопку (.buy-button)

Подсказки:

  • Используйте padding для внутренних отступов
  • Добавьте border-radius для скругления углов
  • Поэкспериментируйте с box-shadow для объема
  • Используйте margin для расстояний между элементами
  • Попробуйте разные font-family и font-weight

Пример решения:

/* Файл style.css */

/* 1. Сброс и базовые стили */
* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}

body {
    font-family: 'Inter', sans-serif;
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    min-height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
    padding: 20px;
}

/* 2. Стили карточки */
.product-card {
    width: 100%;
    max-width: 320px;
    background: white;
    border-radius: 20px;
    overflow: hidden;
    box-shadow: 0 20px 40px rgba(0, 0, 0, 0.1);
    transition: transform 0.3s ease;
}

.product-card:hover {
    transform: translateY(-10px);
}

/* 3. Изображение товара */
.product-card img {
    width: 100%;
    height: 200px;
    object-fit: cover;
}

/* 4. Контент карточки */
.product-card > div {
    padding: 20px;
}

/* 5. Заголовок товара */
.product-title {
    font-size: 1.5rem;
    color: #2d3748;
    margin-bottom: 10px;
    font-weight: 700;
}

/* 6. Описание товара */
.product-description {
    color: #718096;
    font-size: 0.95rem;
    line-height: 1.6;
    margin-bottom: 15px;
}

/* 7. Цена */
.price {
    font-size: 1.8rem;
    color: #2b6cb0;
    font-weight: 800;
    margin-bottom: 20px;
}

/* 8. Кнопка */
.buy-button {
    width: 100%;
    padding: 15px;
    background: linear-gradient(90deg, #667eea 0%, #764ba2 100%);
    color: white;
    border: none;
    border-radius: 10px;
    font-size: 1rem;
    font-weight: 600;
    cursor: pointer;
    transition: all 0.3s ease;
}

.buy-button:hover {
    background: linear-gradient(90deg, #5a67d8 0%, #6b46c1 100%);
    transform: scale(1.02);
}

Шаг за шагом: Создаем первую стилизованную страницу

Шаг 1: Создаем файловую структуру

my-website/
├── index.html
├── style.css
└── images/

Шаг 2: Пишем базовый HTML

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Мой стилизованный сайт</title>
    <link rel="stylesheet" href="style.css">
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
</head>
<body>
    <header class="site-header">
        <h1>Мой первый CSS-сайт</h1>
        <p>Изучаем основы каскадных таблиц стилей</p>
    </header>
    
    <main class="content">
        <section class="intro">
            <h2>Что такое CSS?</h2>
            <p>CSS делает веб-страницы красивыми...</p>
        </section>
        
        <section class="features">
            <div class="feature-card">
                <h3>Селекторы</h3>
                <p>Выбирают элементы для стилизации</p>
            </div>
            <div class="feature-card">
                <h3>Свойства</h3>
                <p>Определяют внешний вид</p>
            </div>
            <div class="feature-card">
                <h3>Значения</h3>
                <p>Конкретные настройки стилей</p>
            </div>
        </section>
    </main>
    
    <footer class="site-footer">
        <p>© 2024 | Изучаем CSS</p>
    </footer>
</body>
</html>

Шаг 3: Создаем CSS файл

/* style.css */

/* 1. Сброс и базовые стили */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Inter', sans-serif;
    line-height: 1.6;
    color: #333;
    background-color: #f9fafb;
}

/* 2. Шапка сайта */
.site-header {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
    text-align: center;
    padding: 60px 20px;
    margin-bottom: 40px;
}

.site-header h1 {
    font-size: 2.5rem;
    margin-bottom: 15px;
    font-weight: 700;
}

.site-header p {
    font-size: 1.2rem;
    opacity: 0.9;
}

/* 3. Основной контент */
.content {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 20px;
}

/* 4. Секция введения */
.intro {
    background: white;
    padding: 40px;
    border-radius: 15px;
    box-shadow: 0 10px 30px rgba(0, 0, 0, 0.08);
    margin-bottom: 40px;
}

.intro h2 {
    color: #2d3748;
    font-size: 1.8rem;
    margin-bottom: 20px;
    border-bottom: 3px solid #667eea;
    padding-bottom: 10px;
}

.intro p {
    font-size: 1.1rem;
    color: #4a5568;
}

/* 5. Карточки особенностей */
.features {
    display: flex;
    gap: 30px;
    flex-wrap: wrap;
    margin-bottom: 40px;
}

.feature-card {
    flex: 1;
    min-width: 250px;
    background: white;
    padding: 30px;
    border-radius: 15px;
    box-shadow: 0 5px 15px rgba(0, 0, 0, 0.05);
    transition: transform 0.3s ease;
}

.feature-card:hover {
    transform: translateY(-5px);
}

.feature-card h3 {
    color: #2d3748;
    font-size: 1.3rem;
    margin-bottom: 15px;
    display: flex;
    align-items: center;
    gap: 10px;
}

.feature-card p {
    color: #718096;
    line-height: 1.6;
}

/* 6. Подвал */
.site-footer {
    background: #2d3748;
    color: white;
    text-align: center;
    padding: 30px;
    margin-top: 60px;
}

.site-footer p {
    opacity: 0.8;
}

/* 7. Адаптивность */
@media (max-width: 768px) {
    .site-header h1 {
        font-size: 2rem;
    }
    
    .features {
        flex-direction: column;
    }
    
    .intro {
        padding: 30px 20px;
    }
}

Шаг 4: Экспериментируем!

Попробуйте изменить:

  • Цвета (поменяйте значения color и background)
  • Размеры (font-size, padding, margin)
  • Шрифты (подключите другие через Google Fonts)
  • Эффекты (box-shadow, border-radius)

Что вы узнали

1. Основные понятия:

  • CSS — язык стилей для оформления HTML
  • Селекторы — выбирают элементы для стилизации
  • Свойства и значения — что и как менять

2. Способы подключения CSS:

  • Встроенные (style атрибут) — плохая практика
  • Внутренние (<style> в <head>) — для простых страниц
  • Внешние (отдельный .css файл) — рекомендуется всегда!

3. Ключевые свойства:

  • Текст: font-family, font-size, color, text-align
  • Блочная модель: width, height, padding, margin, border
  • Важно: box-sizing: border-box

4. Практические навыки:

  • Создание внешнего CSS файла
  • Подключение Google Fonts
  • Стилизация карточек и кнопок
  • Использование box-shadow и border-radius
