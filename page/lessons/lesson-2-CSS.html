Что такое типографика в вебе?

Типографика — это искусство оформления текста. В веб-дизайне правильная работа с текстом делает сайт не только красивым, но и удобным для чтения. Сегодня мы научимся управлять всеми аспектами текста: от выбора шрифта до межбуквенных расстояний.

Простая аналогия: Книга и Веб-страница.

Хорошая книга: удобный шрифт, правильный размер, комфортные междустрочные интервалы

Плохая книга: мелкий нечитаемый текст, слишком яркие цвета, неправильные отступы

Наша задача: сделать веб-страницы такими же удобными для чтения, как хорошие книги

Почему это важно: 95% информации в интернете — это текст. Если его плохо оформить, пользователи просто уйдут с сайта.

Давайте разберем каждое свойство подробно.

Свойство font-family — выбор шрифта

font-family определяет, какой шрифт будет использоваться для текста.

Базовый синтаксис:

selector {
    font-family: "Основной шрифт", запасной шрифт, общее семейство;
}

3 основных семейства шрифтов:

Семейство Особенности Примеры
Serif (засечковые) Есть засечки, традиционный вид Times New Roman, Georgia, Merriweather
Sans-serif (рубленые) Без засечек, современный вид Arial, Helvetica, Roboto, Inter
Monospace (моноширинные) Все символы одинаковой ширины Courier New, Consolas, Source Code Pro

Практические примеры:

/* Базовые системные шрифты */
.body-text {
    font-family: Arial, Helvetica, sans-serif;
}

/* Для длинных текстов (статьи, блоги) */
.article-text {
    font-family: Georgia, 'Times New Roman', serif;
}

/* Для кода */
.code-text {
    font-family: 'Courier New', Courier, monospace;
}

/* Современный дизайн */
.modern-text {
    font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
}

Как подключить Google Fonts:

<!-- В <head> -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;700&family=Merriweather:wght@300;400;700&display=swap" rel="stylesheet">

<!-- В CSS -->
.modern-heading {
    font-family: 'Merriweather', serif;
}

.modern-text {
    font-family: 'Roboto', sans-serif;
}

Важные правила:

  • Всегда указывайте запасные шрифты — если первый не загрузится
  • Названия из нескольких слов в кавычки — "Times New Roman"
  • Сначала специфичные, потом общие — 'Roboto', sans-serif
  • Используйте системные шрифты для скорости — они уже есть у пользователя

Свойства размера: font-size, line-height

font-size — размер шрифта

Единицы измерения:

Единица Описание Пример Когда использовать
px (пиксели) Абсолютный размер font-size: 16px; Точно заданные размеры
em Относительно родителя font-size: 1.5em; Масштабируемые интерфейсы
rem Относительно корня font-size: 1.2rem; Рекомендуется! Единая система
% Проценты от родителя font-size: 150%; Старые браузеры
vw/vh Относительно окна font-size: 3vw; Адаптивные заголовки

Рекомендуемые размеры:

/* Базовая система размеров */
:root {
    font-size: 16px; /* 1rem = 16px */
}

body {
    font-size: 1rem; /* 16px */
}

h1 {
    font-size: 2.5rem; /* 40px */
}

h2 {
    font-size: 2rem; /* 32px */
}

h3 {
    font-size: 1.5rem; /* 24px */
}

p {
    font-size: 1rem; /* 16px */
}

.small-text {
    font-size: 0.875rem; /* 14px */
}

line-height — межстрочный интервал

Определяет расстояние между строками текста.

/* Без единиц (рекомендуется) */
.readable-text {
    line-height: 1.6; /* 1.6 × font-size */
}

/* С единицами */
.code-text {
    line-height: 24px; /* Фиксированное расстояние */
}

/* Примеры использования */
.too-tight {
    line-height: 1;    /* Слишком плотно */
}

.good-for-reading {
    line-height: 1.5;  /* Идеально для чтения */
}

.for-accessibility {
    line-height: 1.8;  /* Для слабовидящих */
}

Правило золотого сечения: line-height должен быть в 1.5-1.8 раза больше font-size.

Свойство font-weight — насыщенность шрифта

Определяет толщину шрифта.

Значения:

.light-text {
    font-weight: 300;   /* Светлый */
}

.normal-text {
    font-weight: 400;   /* Нормальный (обычный) */
}

.medium-text {
    font-weight: 500;   /* Средний */
}

.bold-text {
    font-weight: 700;   /* Жирный */
}

.extra-bold {
    font-weight: 900;   /* Очень жирный */
}

Именованные значения (устаревшие, но работают):

.normal {
    font-weight: normal;  /* = 400 */
}

.bold {
    font-weight: bold;    /* = 700 */
}

Практическое применение:

/* Иерархия заголовков */
h1 {
    font-weight: 700;  /* Самый жирный */
}

h2 {
    font-weight: 600;  /* Чуть менее жирный */
}

h3 {
    font-weight: 500;  /* Средний вес */
}

/* Акценты в текста */
.important {
    font-weight: 600;
    color: #d97706;
}

/* Светлые элементы */
.muted {
    font-weight: 300;
    color: #6b7280;
}

Важно: Не все шрифты поддерживают все значения font-weight. Проверяйте в документации Google Fonts.

Свойство color — цвет текста

Способы задания цвета:

/* По названию (147 стандартных цветов) */
.basic-colors {
    color: red;
    color: blue;
    color: green;
    color: black;
    color: white;
    color: gray;
}

/* HEX (шестнадцатеричный) - самый популярный */
.hex-colors {
    color: #ff0000;      /* Красный */
    color: #00ff00;      /* Зеленый */
    color: #0000ff;      /* Синий */
    color: #333333;      /* Темно-серый */
    color: #f3f4f6;      /* Светло-серый */
}

/* RGB (Red, Green, Blue) */
.rgb-colors {
    color: rgb(255, 0, 0);           /* Красный */
    color: rgb(0, 255, 0);           /* Зеленый */
    color: rgb(0, 0, 255);           /* Синий */
}

/* RGBA (с прозрачностью) */
.rgba-colors {
    color: rgba(255, 0, 0, 0.5);     /* Полупрозрачный красный */
    color: rgba(0, 0, 0, 0.7);       /* Темно-серый с прозрачностью */
}

/* HSL (Hue, Saturation, Lightness) */
.hsl-colors {
    color: hsl(0, 100%, 50%);        /* Красный */
    color: hsl(120, 100%, 50%);      /* Зеленый */
    color: hsl(240, 100%, 50%);      /* Синий */
}

Контрастность текста:

Для хорошей читаемости соблюдайте минимальный контраст:

  • Обычный текст: 4.5:1 (WCAG AA)
  • Крупный текст (от 18px): 3:1

Используйте инструменты: WebAIM Contrast Checker

Свойство text-align — выравнивание текста

Основные значения:

.left-align {
    text-align: left;      /* По левому краю (по умолчанию) */
}

.center-align {
    text-align: center;    /* По центру */
}

.right-align {
    text-align: right;     /* По правому краю */
}

.justify-align {
    text-align: justify;   /* По ширине */
}

Практическое применение:

/* Заголовки обычно по центру */
h1, h2, h3 {
    text-align: center;
}

/* Основной текст по левому краю */
article p {
    text-align: left;
    text-align: justify; /* Для длинных текстов */
}

/* Цитаты часто по центру */
blockquote {
    text-align: center;
    font-style: italic;
}

/* Подписи к изображениям */
figcaption {
    text-align: center;
    font-size: 0.875rem;
    color: #6b7280;
}

/* Кнопки и CTA */
.cta-button {
    text-align: center;
    display: block; /* Для центрирования кнопки */
}

/* Таблицы */
td {
    text-align: right; /* Числа справа */
}

td:first-child {
    text-align: left;  /* Первая колонка слева */
}

text-align для разных языков:

/* Для языков слева направо (русский, английский) */
:lang(ru), :lang(en) {
    text-align: left;
    direction: ltr;
}

/* Для языков справа налево (арабский, иврит) */
:lang(ar), :lang(he) {
    text-align: right;
    direction: rtl;
}

Практика: Создаем типографский макет

Задача: Превратить обычный HTML в профессионально оформленную статью.

Исходный HTML:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Типографский макет</title>
    <link rel="stylesheet" href="typography.css">
</head>
<body>
    <article class="blog-post">
        <header class="post-header">
            <h1 class="post-title">Искусство веб-типографики</h1>
            <p class="post-meta">Автор: Алексей Дизайнер • 15 марта 2024</p>
        </header>
        
        <div class="post-content">
            <p class="lead">Правильная типографика может превратить обычный текст в произведение искусства...</p>
            
            <h2>Выбор шрифта</h2>
            <p>Шрифт — это голос вашего сайта. Как и в реальной жизни, важно выбрать правильный тон...</p>
            
            <blockquote class="quote">
                "Хорошая типографика незаметна. Плохая типографика бросается в глаза."
                <cite>— Неизвестный дизайнер</cite>
            </blockquote>
            
            <h2>Размер и интервалы</h2>
            <p>Размер шрифта и межстрочные интервалы определяют удобство чтения...</p>
            
            <div class="code-example">
                <h3>Пример CSS кода:</h3>
                <pre><code>body {
    font-family: 'Georgia', serif;
    font-size: 18px;
    line-height: 1.6;
    color: #333;
}</code></pre>
            </div>
            
            <h2>Цвет и контраст</h2>
            <p>Цвет текста должен обеспечивать достаточный контраст с фоном...</p>
            
            <ul class="tips-list">
                <li>Используйте темный текст на светлом фоне</li>
                <li>Избегайте чистого черного (#000)</li>
                <li>Проверяйте контрастность</li>
            </ul>
        </div>
        
        <footer class="post-footer">
            <p class="tags">Теги: <span>типографика</span>, <span>веб-дизайн</span>, <span>css</span></p>
        </footer>
    </article>
</body>
</html>

Что нужно сделать в файле typography.css:

  1. Подключить Google Fonts (Georgia для основного текста, Roboto для заголовков)
  2. Стилизовать заголовок статьи
  3. Оформить основной текст с правильными размерами и интервалами
  4. Стилизовать цитату
  5. Оформить блок с кодом
  6. Сделать список с советами
  7. Добавить стили для подвала

Критерии успеха:

  • Читаемый основной текст (16-18px, line-height 1.6-1.8)
  • Четкая иерархия заголовков (h1 > h2 > h3)
  • Правильные цвета (достаточный контраст)
  • Уместное выравнивание
  • Единый стиль всех элементов

Что вы узнали

1. Ключевые свойства текста:

  • font-family — выбор шрифта и семейств
  • font-size — размер с разными единицами измерения
  • font-weight — насыщенность (300-900)
  • color — цвет текста (HEX, RGB, HSL)
  • text-align — выравнивание (left, center, right, justify)

2. Важные концепции:

  • Типографская иерархия — заголовки > подзаголовки > основной текст
  • Контрастность — для доступности и читаемости
  • Межстрочные интервалы — line-height для удобства чтения
  • Система размеров — rem vs px vs em

3. Практические навыки:

  • Подключение Google Fonts
  • Создание читаемых текстовых блоков
  • Оформление цитат и кода
  • Создание адаптивной типографики
  • Работа с цветовыми палитрами для текста

4. Золотые правила веб-типографики:

  • Основной текст: 16-18px, line-height: 1.5-1.8
  • Контраст текста и фона: минимум 4.5:1
  • Используйте rem для размеров
  • Максимум 2-3 шрифта на странице
  • Заголовки должны быть явно крупнее основного текста
