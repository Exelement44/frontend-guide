Что такое HTML?

HTML (HyperText Markup Language) - это не язык программирования, а язык разметки. Его задача - описать структуру и содержание веб-страницы, размечая, где что находится: «вот заголовок», «вот параграф текста», «вот изображение», «вот ссылка».

Простая аналогия: Строительство дома.

  • HTML - это фундамент, несущие стены, окна и двери. Он создает каркас и определяет, где будут расположены основные элементы. Без этого дом просто не может существовать.
  • CSS - это отделка: цвет обоев, стиль дверей, форма оконных рам, ландшафтный дизайн. CSS отвечает за внешний вид и красоту.
  • JavaScript — это инженерия: выключатели света, лифт, система отопления, дверной звонок. JavaScript делает страницу интерактивной и «живой».

Базовая структура HTML-документа

Каждый HTML-документ следует единому шаблону. Запомнив его однажды, вы сможете создавать новые страницы за секунды.

Вот минимальный, но полностью рабочий код:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Мой первый сайт</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая HTML-страница.</p>
</body>
</html>

Давайте разберем каждую строку по порядку.

<!DOCTYPE html> - Объявление типа документа

Это самая первая и обязательная строка. Ее можно сравнить с указанием «Это инструкция по сборке мебели IKEA версии 5». Она сообщает браузеру: «Работай в самом современном режиме обработки HTML (HTML5)». Без этой строки браузер может переключиться в «режим совместимости» со старыми стандартами, и ваша страница может отображаться некорректно.

Раздел <head>: Инструкция для браузера и поисковиков

Тег <head> — это служебный раздел. Ничего из того, что находится внутри <head>, пользователь напрямую на странице не видит (кроме заголовка вкладки). Зато эту информацию очень внимательно читает браузер и поисковые системы.

  • <html lang="ru"> : Атрибут lang указывает язык страницы. Это важно для программ чтения с экрана (скринридеров) и для поисковиков.
  • <meta charset="UTF-8"> : Самая важная мета-информация. Устанавливает кодировку символов. UTF-8 включает поддержку кириллицы (русских букв) и практически всех языков мира. Без этой строчки вместо текста могут появиться «кракозябры».
  • <meta name="viewport" ...> : Ключевой тег для адаптивности. Он сообщает браузеру, что ширину страницы нужно подстраивать под ширину экрана устройства (телефона, планшета). Это основа современных мобильных версий сайтов.
  • <title>Мой первый сайт</title> : Заголовок страницы. Он отображается на вкладке браузера, сохраняется в закладках и является одним из главных факторов для SEO (поисковой оптимизации).

Раздел <body> : Пространство для контента

Тег <body> - это видимая часть страницы, ради которой все и затевается. Все, что вы напишете или разместите внутри <body>, увидят пользователи: текст, картинки, видео, кнопки, меню.

Здесь начинается творчество и работа с контентом.

Первые и самые важны теги

Давайте познакомимся с четырьмя китами, на которых держится большинство страниц.

<h1> ... <h6> - Заголовки (Headings)

Это иерархические заголовки. <h1> - самый главный, заголовок всей страницы (он обычно один). Далее идут подзаголовки: <h2>, <h3> и так до <h6>.

  • Важно для структуры: Заголовки создают «оглавление» вашей страницы.
  • Критически важно для SEO: Поисковые системы с их помощью понимают, о чем ваш контент.
<h1>Название статьи (главная мысль)</h1>
<h2>Глава 1. Введение</h2>
<h3>1.1. Предыстория проблемы</h3>
<h2>Глава 2. Основная часть</h2>

<p> - Параграф (Paragraph)

Тег для обычного текста. Каждый новый параграф логически отделяется от другого.

<p>Это первый абзац моего текста. Здесь я рассказываю о чем-то важном.</p>
<p>А это второй абзац. Он начинается с новой строки, и между абзацами есть небольшой отступ.</p>

<a> - Ссылка (Anchor / Якорь)

Создает гиперссылки для перехода на другие страницы или сайты.

  • Атрибут href - это адрес, куда ведет ссылка (обязательный).
  • Атрибут target="_blank" - необязательный, но полезный. Он открывает ссылку в новой вкладке.
<p>Посетите <a href="https://yandex.ru" target="_blank">Яндекс</a> для поиска информации.</p>

<img> - Изображение (Image)

Тег для вставки картинок. Это одиночный тег (не требует закрывающего).

  • Атрибут src - путь к файлу изображения (URL или имя файла в папке).
  • Атрибут alt - альтернативный текст. Это обязательно! Он появляется, если картинка не загрузилась, и его «озвучивают» скринридеры для незрячих людей.
<img src="my_cat.jpg" alt="Фотография моего рыжего кота, спящего на диване">

Практика: Создаем свою первую страницу

Теория без практики ничего не стоит. Давайте создадим реальный файл.

Шаг 1: Создание файла

  1. Откройте любой текстовый редактор (подойдет даже «Блокнот», но лучше VS Code, Sublime Text или Pulsar).
  2. Создайте новый файл
  3. Скопируйте и вставьте код ниже
  4. Сохраните файл с именем index.html. Важно: расширение должно быть именно .html, а не .txt.
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Мое творение</title>
</head>
<body>
    <h1>Добро пожаловать в мой мир!</h1>
    <h2>Немного о себе</h2>
    <p>Меня зовут [Ваше Имя]. Я начинающий веб-разработчик и сегодня создал свою первую страницу.</p>
    <p>Это невероятно увлекательно — видеть, как код превращается в то, что можно увидеть в браузере!</p>

    <h2>Мои увлечения</h2>
    <p>Я люблю читать, путешествовать и изучать новые технологии. А еще я большой фанат фильмов Marvel!</p>

    <h2>Полезный ресурс</h2>
    <p>Больше информации о веб-разработке можно найти на <a href="https://developer.mozilla.org/ru/" target="_blank">MDN Web Docs</a> - это лучшая документация для разработчиков.</p>

    <h2>Мое вдохновение</h2>
    <img src="https://via.placeholder.com/600x400" alt="Пример изображения-заглушки. Здесь мог бы быть красивый пейзаж.">
    <p><small>Для своей страницы вы можете заменить этот тег на `<img src="ваша_фото.jpg" alt="описание">` и положить картинку в ту же папку, что и index.html.</small></p>
</body>
</html>

Шаг 2: Просмотр в браузере

Найдите файл index.html в папке и просто дважды щелкните по нему. Он автоматически откроется в вашем стандартном браузере (Chrome, Firefox, Edge и т.д.). Вуаля, вы только что запустили свой веб-сайт!

Шаг 3: Эксперименты

Самое важное - начать изменять код. Попробуйте прямо сейчас:

  1. Измените текст внутри <title> и обновите страницу в браузере (F5). Посмотрите, что изменилось.
  2. Добавьте еще один параграф <p> с любым текстом.
  3. Создайте ссылку на свой любимый сайт.
  4. Найдите в интернете картинку, сохраните ее в ту же папку, что и index.html, и загрузите, указав ее имя в атрибуте src тега <img> (например, src="nature.jpg").

Что вы узнали

  1. HTML - это язык разметки, структурный скелет веб-страницы.
  2. Каждый документ начинается с <!DOCTYPE html> и имеет обязательную структуру <html> → <head> + <body>.
  3. Раздел <head> содержит служебную информацию (кодировку, заголовок вкладки, настройки для мобильных).
  4. Раздел <body> - это контейнер для всего видимого контента.
  5. Вы познакомились с ключевыми тегами: заголовки <h1>-<h6>, параграф <p>, ссылка <a href="...">, изображение <img src="..." alt="...">.
  6. Вы создали, сохранили и открыли свой первый HTML-файл!
