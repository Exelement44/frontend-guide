Что такое типы данных?

Типы данных — это разные виды информации, с которыми работает программа. Как в жизни есть разные форматы информации (текст, числа, да/нет), так и в программировании.

Простая аналогия: Школьный рюкзак.

Представьте, что переменные — это отделения в рюкзаке:

  • Числа — как учебники по математике (можно складывать, вычитать)
  • Строки — как тетради с сочинениями (текст, который можно читать и соединять)
  • Булевы значения — как галочки в дневнике (✓ сделано, ✗ не сделано)

Каждый тип данных обрабатывается по-своему, и JavaScript должен понимать, с чем он работает.

Основные типы данных

1. Числа (Numbers) — для математики

Числа используются для подсчетов, вычислений, измерений.

let age = 15;          // Целое число
let price = 99.99;     // Десятичная дробь
let temperature = -5;  // Отрицательное число

Математические операции:

let a = 10;
let b = 3;

console.log(a + b);   // 13 (сложение)
console.log(a - b);   // 7 (вычитание)
console.log(a * b);   // 30 (умножение)
console.log(a / b);   // 3.333... (деление)
console.log(a % b);   // 1 (остаток от деления)

Особые числа:

console.log(10 / 0);   // Infinity (бесконечность)
console.log(-10 / 0);  // -Infinity
console.log(0 / 0);    // NaN (Not a Number - не число)

2. Строки (Strings) — для текста

Строки — это текст в кавычках. Используются для слов, сообщений, имен.

let name = "Анна";       // Двойные кавычки
let city = 'Москва';     // Одинарные кавычки
let message = `Привет`;  // Обратные кавычки

Операции со строками:

let firstName = "Иван";
let lastName = "Петров";

// Конкатенация (соединение строк)
let fullName = firstName + " " + lastName;
console.log(fullName); // "Иван Петров"

// Длина строки
console.log(fullName.length); // 11 (символов)

Шаблонные строки (обратные кавычки):

let age = 15;
let text = `Мне ${age} лет`; // Удобная вставка переменных
console.log(text); // "Мне 15 лет"

3. Булевы значения (Boolean) — да/нет

Всего два значения: true (правда) и false (ложь). Используются для проверок условий.

let isStudent = true;     // Да, я ученик
let hasHomework = false;  // Нет, домашку не сделал
let isAdult = false;      // Нет, не совершеннолетний

Определение типа: typeof

Узнаем, какой тип данных хранится в переменной:

console.log(typeof 42);           // "number"
console.log(typeof "Привет");     // "string"
console.log(typeof true);         // "boolean"
console.log(typeof undefined);    // "undefined"

Преобразование типов

JavaScript иногда сам преобразует типы:

// Строка + число = строка
console.log("Возраст: " + 15); // "Возраст: 15"

// При умножении/делении строки преобразуются в числа
console.log("10" * 2);  // 20
console.log("10" / 2);  // 5

// Явное преобразование
let str = "123";
let num = Number(str);  // Преобразуем строку в число
console.log(typeof num); // "number"

Операторы сравнения

Операторы сравнения возвращают true или false. Используются для принятия решений в программе.

1. Операторы равенства

let a = 5;
let b = 10;

console.log(a == 5);   // true (равно)
console.log(a != 5);   // false (не равно)
console.log(a == b);   // false
console.log(a != b);   // true

Важно: == сравнивает значения, === сравнивает и значения, и типы:

console.log(5 == "5");   // true (значения одинаковые)
console.log(5 === "5");  // false (типы разные: number vs string)

Правило: Используйте === и !== — они строже и надежнее.

2. Операторы сравнения

let score = 85;

console.log(score > 80);   // true (больше)
console.log(score < 80);   // false (меньше)
console.log(score >= 85);  // true (больше или равно)
console.log(score <= 84);  // false (меньше или равно)

3. Логические операторы

Объединяют несколько условий:

let age = 15;
let hasTicket = true;

// И (AND) — оба условия должны быть true
console.log(age >= 12 && hasTicket); // true (и возраст подходит, и билет есть)

// ИЛИ (OR) — хотя бы одно условие true
console.log(age >= 18 || hasTicket); // true (билет есть, хотя возраст не подходит)

// НЕ (NOT) — инвертирует значение
console.log(!hasTicket); // false (не имеет билета = false)

Практика: Задание 1 — Калькулятор оценок с проверками

Цель: Создать программу, которая проверяет оценки ученика.

console.log('=== ПРОВЕРКА УСПЕВАЕМОСТИ ===');

// Данные ученика
const studentName = "Мария";
const mathGrade = 4;
const russianGrade = 5;
const englishGrade = 3;

// Выводим информацию
console.log(`Ученик: ${studentName}`);
console.log(`Математика: ${mathGrade}`);
console.log(`Русский: ${russianGrade}`);
console.log(`Английский: ${englishGrade}`);

// Проверки
const hasExcellentMath = mathGrade === 5;
const hasGoodRussian = russianGrade >= 4;
const hasProblemsWithEnglish = englishGrade < 4;

console.log('='.repeat(30));
console.log('Отличник по математике?', hasExcellentMath);
console.log('Хорошист по русскому?', hasGoodRussian);
console.log('Проблемы с английским?', hasProblemsWithEnglish);

// Общая проверка
const isGoodStudent = mathGrade >= 4 && russianGrade >= 4;
console.log('Хороший ученик в целом?', isGoodStudent);

// Можно ли на олимпиаду?
const canGoToOlympiad = mathGrade === 5 || russianGrade === 5;
console.log('Можно на олимпиаду?', canGoToOlympiad);

Практика: Задание 2 — Проверка возраста для кинотеатра

Цель: Создать программу, которая проверяет, можно ли смотреть фильм.

console.log('=== ПРОВЕРКА ВОЗРАСТА ДЛЯ КИНО ===');

// Данные посетителя
let visitorName = "Алексей";
let visitorAge = 14;
let hasParent = true;
let movieRating = 16; // Возрастное ограничение фильма

console.log(`Посетитель: ${visitorName}`);
console.log(`Возраст: ${visitorAge} лет`);
console.log(`Рейтинг фильма: ${movieRating}+`);
console.log(`С родителем? ${hasParent ? 'Да' : 'Нет'}`);

// Проверки
let canWatchAlone = visitorAge >= movieRating;
let canWatchWithParent = visitorAge >= 12 && hasParent;
let canWatchAtAll = canWatchAlone || canWatchWithParent;

console.log('='.repeat(30));
console.log('Может смотреть один?', canWatchAlone);
console.log('Может смотреть с родителем?', canWatchWithParent);
console.log('Может смотреть вообще?', canWatchAtAll);

// Детальный вывод
if (canWatchAlone) {
    console.log('Можете проходить, возраст подходит!');
} else if (canWatchWithParent) {
    console.log('Только в сопровождении взрослого');
} else {
    console.log('Извините, вам рано смотреть этот фильм');
}

// Дополнительные проверки
console.log('='.repeat(30));
console.log('Проверка типов данных:');
console.log('Тип возраста:', typeof visitorAge);
console.log('Тип имени:', typeof visitorName);
console.log('Тип "с родителем":', typeof hasParent);

// Преобразование типов
let ageString = String(visitorAge); // Число в строку
console.log('Возраст как строка:', ageString, 'тип:', typeof ageString);

let parentString = String(hasParent);
console.log('"С родителем" как строка:', parentString);

Что вы узнали

  1. Три основных типа данных:
    • Числа (number) — для вычислений: 10, 3.14, -5
    • Строки (string) — для текста: "Привет", 'Мир'
    • Булевы значения (boolean) — для условий: true, false
  2. Операции с числами:
    • +, -, *, /, % (остаток от деления)
    • Особые значения: Infinity, NaN
  3. Операции со строками:
    • + для соединения строк
    • .length для длины строки
    • Шаблонные строки с ${переменная}
  4. Операторы сравнения (возвращают true/false):
    • == и === (равно) — используйте ===
    • != и !== (не равно) — используйте !==
    • >, <, >=, <= (больше/меньше)
  5. Логические операторы:
    • && (И) — оба условия true
    • || (ИЛИ) — хотя бы одно условие true
    • ! (НЕ) — инвертирует значение
  6. Важные функции:
    • typeof — узнать тип данных
    • Number() — преобразовать в число
    • String() — преобразовать в строку
