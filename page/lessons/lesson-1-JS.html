Что такое JavaScript?

JavaScript — это язык программирования, который делает веб-страницы "живыми". Если HTML — это скелет страницы, а CSS — её внешний вид, то JavaScript — это мышцы и мозг, которые заставляют страницу реагировать на ваши действия.

Простая аналогия: Умный дом.

Представьте себе дом:

  • HTML — фундамент, стены, окна, двери (структура)
  • CSS — обои, краска, мебель (внешний вид)
  • JavaScript — выключатели света, термостат, дверной звонок, система безопасности (интерактивность)

Когда вы нажимаете кнопку на сайте, заполняете форму, видите анимацию или всплывающее окно — всё это работает на JavaScript. Это язык, который понимает браузер и выполняет прямо на вашем компьютере.

Как подключить JavaScript к странице

Есть три основных способа подключить JavaScript:

Способ 1: Внутри тега <script> (внутри HTML)

<!DOCTYPE html>
<html>
<head>
    <title>Моя страница</title>
</head>
<body>
    <h1>Привет!</h1>
    
    <script>
        // Здесь пишем JavaScript код
        alert('Страница загружена!');
    </script>
</body>
</html>

Способ 2: Внешний файл (рекомендуется)

<!DOCTYPE html>
<html>
<head>
    <title>Моя страница</title>
</head>
<body>
    <h1>Привет!</h1>
    
    <!-- Подключаем внешний файл -->
    <script src="script.js"></script>
</body>
</html>

Файл script.js:

// Код в отдельном файле
alert('Привет из внешнего файла!');

Способ 3: В <head> с атрибутом defer

<head>
    <script src="script.js" defer></script>
</head>

defer означает: "загрузи скрипт, но выполни его только после загрузки всей страницы".

Правило для начинающих: Используйте Способ 2 — отдельный файл script.js. Это аккуратнее и проще для организации кода.

Первая команда: console.log() — ваш лучший друг

console.log() — это способ "поговорить" с консолью браузера. Консоль — это специальное окно, где можно видеть сообщения от JavaScript.

console.log('Привет, консоль!');
console.log(2 + 2);
console.log('Меня зовут:', 'Анна');

Как открыть консоль в браузере:

  1. Нажмите F12 или правой кнопкой → "Исследовать элемент"
  2. Перейдите на вкладку Console
  3. Там вы увидите результаты console.log()

Зачем это нужно?

  • Проверять, работает ли ваш код
  • Видеть результаты вычислений
  • Находить ошибки
  • Отлаживать (исправлять) программу

Примеры использования:

console.log('Начало программы');
console.log('2 + 3 =', 2 + 3);
console.log('Текущее время:', new Date());

Переменные: Ящики для хранения данных

Переменная — это как коробка с названием, в которую можно положить что угодно (число, текст, список) и потом достать по названию.

Создание переменных: let и const

let — для переменных, которые МОГУТ меняться

let age = 15;     // Создали переменную age со значением 15
age = 16;         // Изменили значение на 16 (можно!)
console.log(age); // Покажет 16

const — для переменных, которые НЕ МОГУТ меняться (константы)

const birthYear = 2008;  // Создали константу
birthYear = 2009;        // ОШИБКА! Нельзя изменить константу

Правила именования переменных:

  • Могут содержать буквы, цифры, _, $
  • Не могут начинаться с цифры
  • Чувствительны к регистру (age и Age — разные переменные)
  • Нельзя использовать зарезервированные слова (let, const, if и т.д.)

Хорошие имена:

let userName = "Анна";    // понятно
let user_age = 15;        // с подчеркиванием
let isOnline = true;      // булево значение

Плохие имена:

let a = 10;               // что означает "a"?
let 1stPlace = "первый";  // начинается с цифры
let let = 5;              // зарезервированное слово

Практика: Простая программа

Задание: Знакомство с JavaScript

Шаг 1: Создаем файлы

index.html:

<!DOCTYPE html>
<html>
<head>
    <title>Мой первый JS</title>
</head>
<body>
    <h1>Проверяем JavaScript</h1>
    <script src="script.js"></script>
</body>
</html>

script.js:

console.log('=== Моя первая программа ===');

// Создаем переменные
const myName = "Алексей";
let myAge = 14;

// Выводим в консоль
console.log('Меня зовут:', myName);
console.log('Мне лет:', myAge);

// Пробуем изменить переменную let
myAge = 15;
console.log('Ой, скоро будет:', myAge);

// Это вызовет ошибку (раскомментируйте, чтобы увидеть):
// const school = "Школа №5";
// school = "Другая школа"; // ОШИБКА!

Шаг 2: Открываем консоль

  1. Откройте index.html в браузере
  2. Нажмите F12
  3. Перейдите на вкладку "Console"
  4. Увидите:
=== Моя первая программа ===
Меня зовут: Алексей
Мне лет: 14
Ой, скоро будет: 15

Шаг 3: Экспериментируем

Попробуйте:

  • Изменить значения переменных
  • Добавить новые переменные
  • Поменять текст в console.log()
  • Попробовать изменить const (увидите ошибку)

Что вы узнали

  1. JavaScript делает страницы интерактивными
  2. Подключается через <script src="file.js"></script>
  3. console.log() выводит информацию в консоль (открывается F12)
  4. Переменные хранят данные:
    • const — для постоянных значений
    • let — для изменяемых значений
  5. Консоль браузера — ваш основной инструмент для проверки кода
